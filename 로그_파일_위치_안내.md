## 👀 로그 확인 방법

### WSL에서 확인

```bash
# 프로젝트 디렉토리로 이동
cd /home/sth0824/ansible

# 최근 로그 보기
tail -20 api_auto_update.log

# 실시간 로그 보기
tail -f api_auto_update.log

# 전체 로그 보기
cat api_auto_update.log
```

### Windows에서 확인

1. **파일 탐색기에서:**
   - 주소창에 `\\wsl$\Ubuntu\home\sth0824\ansible` 입력
   - `api_auto_update.log` 파일 열기

2. **VS Code에서:**
   - 프로젝트 루트에서 `api_auto_update.log` 파일 열기

3. **PowerShell에서:**
   ```powershell
   wsl cat /home/sth0824/ansible/api_auto_update.log
   ```
