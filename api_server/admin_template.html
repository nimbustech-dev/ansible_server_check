양식 안내

컬럼 순서는 상관없습니다. 첫 행에 헤더명만 있으면 되며, 한글(호스트명·비밀번호·SSH사용자 등) 또는 영문(name·ip·password·ssh_user 등) 모두 인식합니다.

수정: name 또는 ip로 기존 서버를 찾아 password·ssh_user만 갱신. 셀 비우면 기존 값 유지. / 신규 등록: 매칭되는 서버가 없고 name·ip가 둘 다 있으면 새 서버로 추가.

필수: name(호스트명), ip. 선택: password(비밀번호), ssh_user(SSH사용자), os_type(OS유형), ssh_port(SSH포트), memo(비고), ssh_auth_type(인증방식), ssh_key_path(키경로). .xlsx만 지원.
