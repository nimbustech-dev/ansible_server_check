  - 상세 정보 확인 플로우

  - 접근성 문제 (키보드 네비게이션, 스크린 리더)
  - 성능 이슈 (로딩 속도, 차트 렌더링)
  - 브라우저 호환성

- [ ] **UX 문제점 리스트 작성**

### 3. 디자인 시스템 기초 설계 ✅ 필수

#### 3.1 디자인 토큰 정의
- [ ] **색상 팔레트 확정**
  - Primary 색상 (현재: #1e3a8a, #3b82f6)
  - Success/Error/Warning 색상
  - Neutral 색상 (배경, 텍스트, 보더)
  - 상태별 색상 (정상/경고/위험)

- [ ] **타이포그래피 시스템**
  - 제목 계층 (H1~H6)
  - 본문 텍스트 크기
  - 폰트 패밀리 (현재: 시스템 폰트 스택)

- [ ] **간격 시스템 (Spacing)**
  - 기본 단위 (4px, 8px 기준)
  - 컴포넌트 간격 규칙

- [ ] **그림자 및 효과**
  - Elevation 레벨별 그림자
